Requirements

Task 1

Home page - simple home page which will contain welcome message with 2 links inside that will lead to registration page and download mobile page

Task 2

Mobile page - page contains information about our mobile application with 2 icons on the bottom of the page. The icons will lead to the Apple App store (https://www.apple.com/app-store/) and to the Google Play store (https://play.google.com/store/apps)

Task 3

Registration page - should contains 4 input field for username, email, password and confirm password and 2 button for registration and cancellation of the process.

Design for desktop - check the image for reference - the label should on the top of the inputs. Inputs width should be 270px. Action buttons should be in single line with 15px gap between them and align with the input width. Register button should follow global style guide and should be primary button, cancel button should follow global style guide and should be secondary button

Design for mobile (screen under 1024px) - check the image for reference - The page should have 15px spacing on each side. The label should on the top of the inputs. Inputs width should be 100% of remaining size after we set page spacing. Action buttons should be in two lines with 15px gap between them and should take the same size as inputs. Register button should follow global style guide and should be primary button, cancel button should follow global style guide and should be secondary button

Field validations - validation message should appear under the input in red color when user type any symbol in to the field

  • Username
    1. Required
    2. Must start with letter
    3. Should contains only letters or numbers
    4. Minimum length of 6 symbols
  • Email
    1. Required
    2. Must be valid email address
  • Password
    1. Required
    2. Should contains only letters or numbers
    3. Minimum length of 8 symbols
  • Confirm Password
    1. Required
    2. Should be equal to password field

Action buttons

Register button - should be always enabled. When it clicked and there is no validation errors, the form should be cleared and green successful message for account creation should appear on the top of the form. If there is any validation error, single field error will appear on the top of the form in red color.

Cancel button - should be always enabled. When the user clicks it, it should lead to home page.
